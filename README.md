# Autoservicio ABC 2 - Sistema de Gestión de Autopartes

Proyecto universitario de desarrollo web para un **autoservicio de autopartes** ubicado en La Chorrera, Panamá Oeste.

## 📋 Descripción del Proyecto

**Autoservicio ABC 2** es una solución web completa que combina una página pública informativa con un panel administrativo para gestionar el inventario y las ventas de una tienda física de autopartes.

El sistema permite a los clientes consultar productos y precios, mientras que el personal de la tienda puede gestionar eficientemente el stock, ventas y pedidos.

## ✨ Funcionalidades Principales

### 🌐 Parte Pública (Clientes)

- **Catálogo de productos** con búsqueda y filtros
- **Sobre Nosotros**
- **Información de Pagos** (Efectivo, Tarjeta Visa/Mastercard, Yappy)
- **Información de Entrega** (Solo retiro en tienda)
- **Política de Devoluciones, Reclamos y Garantías**
- **Registro de cuenta** y **Inicio de sesión**
- **Pedidos online con retiro en tienda** (Click & Collect)

### 🔐 Panel de Administración (Solo para personal autorizado)

- **Dashboard**: Resumen de ventas del día, stock bajo y estadísticas
- **Gestión de Productos**: Crear, editar y eliminar productos
- **Gestión de Inventario**: Registrar entradas y ajustes de stock
- **Registro de Ventas**: Ventas rápidas en tienda con actualización automática de stock
- **Clientes**: Gestión de clientes registrados desde la web
- **Proveedores**: Control de proveedores y compras
- **Pedidos para Retiro**: Ver y marcar como entregados los pedidos realizados en la web
- **Reportes**: Estadísticas básicas de ventas e inventario
